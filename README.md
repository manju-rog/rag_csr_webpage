# AI Applications Suite

A premium suite featuring RAG and CSR Generator tools powered by advanced language models.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Create a `.env.local` file and set your `API_KEY`:
   ```
   API_KEY="your-api-key-here"
   ```
3. Run the app:
   `npm run dev`
